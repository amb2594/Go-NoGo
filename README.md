# Go-NoGo Homework 
This is a tool to be used as a proxy for inhibition/disinhibiton measurement-- additionally attention, memory, and other components of cognition

This repository includes: 
  (1) stimuli folder
  (2) Go/NoGo task code

To run this experiment, you must 
  (1) download the food stimuli folder 
  (2) download the code file
  (3) download psychopi
  (4) open the code in psychopi 
  (5) change the location of the food stumili folder to its respective location in the Go/NoGo task code
  (5) run the updated personalized task code of psychopi

Unfortunetly, due to this being a homework assignment, I am not able to upload all of the stimuli folder. Therefor, this is not an actual reliable task to be used for any validated measurement.
